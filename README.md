### Submodules:

1. `env` funcs:

* MustBePresented(envVars ...string)

> Description
> <br>
> Input: env variables names
> <br>
> Output: -
> <br>
> Panic if at least one given env variable is not presented
> <br>
> Example: MustBePresented("VAR1", "VAR2")
